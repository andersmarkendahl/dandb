Dan Data Base

A database that contains a family tree that starts with Dan

Instructions

Search with empty string to fetch whole database and see available fields.

Text strings must be enclosed with apostrophes ('').

Operands AND, OR and NOT are supported.

Examples:

  • firstName='Anders'
  • firstName='Marika' OR middleName='Jan'
  • firstName='Jan' OR (lastName='Markendahl' AND gender='M')
