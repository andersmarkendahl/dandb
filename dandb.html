DanDB

A database that starts with Dan

Search

Empty string to fetch whole database.

Text must be enclosed with apostrophes ('').

Operands AND, OR and NOT are supported.

Examples:

  • firstName='Anders'
  • firstName='Marika' OR middleName='Jan'
  • firstName='Jan' OR (lastName='Markendahl' AND gender='M')

Generate Tree

Empty input generates full tree.

Insert id of person to generate subtree.
